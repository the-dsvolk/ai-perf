Connection Legend

NV18 (NVLink)

Highest-speed direct GPU-to-GPU connection.

PIX (PCIe Local)

Fastest GPU-to-NIC path on the same local fabric.

NODE (PCIe NUMA)

Fast path to a NIC on the same NUMA node.

SYS (System Cross-Connect)

Slowest path; data crosses between CPUs.
